\chapter*{{\khmoul бһўбһҸбҹ’бһҗбһ”бһ‘бһҹбһ„бҹ’бһҒбҹҒбһ”}}
\addcontentsline{toc}{chapter}{{\khmoul бһўбһҸбҹ’бһҗбһ”бһ‘бһҹбһ„бҹ’бһҒбҹҒбһ”}}

\begin{khmer}

бһ“бһ·бһҖбҹ’бһҒбҹҒбһ”бһ”бһ‘бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶ бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһ“бҹғ \textbf{\textit{kfc\_procedure}} бһҠбҹӮбһӣбһҮбһ¶бһҖбһүбҹ’бһ…бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Python бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбҹҖбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹӮбһ” clusterwise supervised learning бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбһ¶бһҳбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ COBRA-based ensemble aggregationбҹ” бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһ бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ“бҹ… Research and Data Analytics Laboratory бһ“бҹғ Department of Applied Mathematics and Statistics бһ“бҹ… Institute of Technology of Cambodiaбҹ” бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹҒбҹҮбһӮбһәбһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ бһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһё бһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„ clusterwise learning, consensus aggregation бһ“бһ·бһ„ GradientCOBRA бһұбҹ’бһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҖбҹ’бһҡбһ”бһҒбһҺбҹ’бһҢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһјбһҢбһ»бһӣбҹ”

бһҖбһүбҹ’бһ…бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһЎбһҫбһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ…бҹҶбһ“бһҪбһ“бһ–бһёбһҡбҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ„бһ‘бһёбһҳбһҪбһҷбһӮбһә \textbf{\textit{KFCProcedure}} бһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ clusterwise learning бһҮбһ¶бһ”бһёбһҮбҹҶбһ бһ¶бһ“бҹ” K-step бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбһғбһҫбһүбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ cluster бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ divergence-based clusteringбҹ” F-step бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ cluster бһ“бһёбһҳбһҪбһҷбҹ—бҹ” C-step бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ–бһёбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ divergence-based бһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһ”бһ¶бһ“бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ„бһ‘бһёбһ–бһёбһҡбһӮбһә \textbf{\textit{GradientCOBRA}} бһҠбҹӮбһӣбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбһ¶бһҳ COBRA бһҡбһҪбһҳбһҳбһ¶бһ“ \textbf{\textit{GradientCOBRA}} бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ regression, \textbf{\textit{MixCOBRARegressor}} бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ‘бһ¶бҹҶбһ„ input-space бһ“бһ·бһ„ prediction-space бһ“бһ·бһ„ \textbf{\textit{CombinedClassifier}} бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ classification aggregationбҹ”

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҸбһ¶бһҳбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҳбҹүбһјбһҢбһ»бһӣ бһҠбҹ„бһҷбһ”бҹҶбһ”бҹӮбһҖбһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ base estimators, normalization, distance computation, kernel weighting, optimization, loss functions, clustering, local learning бһ“бһ·бһ„ prediction aggregationбҹ” бһҖбҹ’бһҡбһ”бһҒбһҺбҹ’бһҢбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһұбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ–бһ„бҹ’бһҡбһёбһҖ бһ“бһ·бһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ machine learning бһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Pythonбҹ”

бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ supervised learning бһ…бҹҶбһ“бһҪбһ“бһ”бҹ’бһҡбһ¶бҹҶбһ–бһёбһҡ бһҠбҹӮбһӣбһҡбһҪбһҳбһҳбһ¶бһ“ classification бһ“бһ·бһ„ regression tasksбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ classification experiments бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Heart Disease, Diabetes, Loan Prediction бһ“бһ·бһ„ Cyber Security Attacksбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ regression experiments бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Abalone, California Housing бһ“бһ·бһ„ Wind Turbineбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ repeated train--test splits бһӣбһҫ random seeds бһ…бҹҶбһ“бһҪбһ“бһ”бҹ’бһҡбһ¶бҹҶбҹ” бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– classification бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢбһҠбҹ„бһҷ Accuracy, Precision, Recall бһ“бһ·бһ„ F1-score бһҒбһҺбҹҲбһҠбҹӮбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– regression бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢбһҠбҹ„бһҷ Mean Absolute Percentage Error, Root Mean Squared Error бһ“бһ·бһ„ coefficient of determinationбҹ”

бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҖбһ¶бһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҖбҹ’бһҡбһ”бһҒбһҺбҹ’бһҢбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӣбһҫбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘ prediction tasks бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” \textbf{\textit{KFCClassifier}} бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ’бҹ’бһҷбһҳбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһӣбһҫбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Heart Disease бһ“бһ·бһ„бһ“бҹ…бһҸбҹӮбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бҹ’бһҡбһҖбһҪбһҸбһ”бҹ’бһҡбһҮбҹӮбһ„бһӣбһҫбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ classification бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһңбһ¶бһҳбһ·бһ“бһ”бһ¶бһ“бһӣбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣ baseline бһҠбҹӮбһӣбһҒбҹ’бһӣбһ¶бҹҶбһ„бһ”бҹҶбһ•бһ»бһҸбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбһ“бҹ„бҹҮбһ‘бҹҒбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ regression, \textbf{\textit{KFCRegressor}} бһҮбһ¶бһҳбһҪбһҷ \textbf{\textit{GradientCOBRA}} combiner бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбһӣбһҫбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ regression бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
